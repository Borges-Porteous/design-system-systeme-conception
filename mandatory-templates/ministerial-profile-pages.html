Ministerial profile pages for Canada.ca

From Treasury Board of Canada Secretariat

Mandatory

  • A ministerial profile page is the landing page for a minister in the Government of Canada, which includes biographical information along with links to initiatives and recent activities related to the minister.
  • Ministerial profiles:
    • provide a consistent biographical overview of a minister, along with timely information about recent activities and announcements
    • allow people to easily understand the minister’s role in government
    • enable people to discover institutions under the minister’s responsibility
    • provide images and videos displaying ministerial activities

On this page

  • When to use this template
  • How to use this template:
    • Profile page
    • Parliamentary secretary modification
  • Working examples
  • User navigation

When to use this template

  • the ministerial profile page template must be used only once as the single source for information about each minister on Canada.ca
  • it must be used to present heads of quasi-judicial arm’s-length institutions using the following modifications:
    • a link from this template to the ministers index on Canada.ca is not required
    • do not display the riding details components
    • the head’s title or role is mandatory and will be displayed in the position of the ministerial role, without any other details
    • the head’s name is mandatory and will be displayed in the position of the Minister’s name
  • it must be used for parliamentary secretaries available through a “Parliamentary Secretaries” tab on the ministers index on Canada.ca

How to use this template

Profile page

1a: Minister’s name

Mandatory

Provides the name of the minister

  • Content
    • must follow mandatory text format: The Honourable [minister’s first and last names], MP
  • Presentation
    • the minister’s name must be a unique H1
    • must be the first component on the page

2: Minister’s picture

Mandatory

Presents a high-quality image of the minister

  • Content
    • must be the minister’s approved headshot
  • Presentation
    • appears at the top of the page, to the left of the biographical overview
    • the image must be linked to a high-resolution image of the minister
    • no text may be embedded in the image
    • go to the Canada.ca GitHub page for image sizing details

3: Ministerial role(s) and riding

Mandatory

Lists the official role(s) of the minister, with links to lead institutional profile, riding information, mandate letter, briefing book, and ministerial appointments where applicable

  • Content
    • mandate letter, briefing book and ministerial appointments must be presented if content is available
    • the name of the riding is hyperlinked to the appropriate riding page from www.parl.gc.ca:
    • it must follow the following text format, each with the corresponding hyperlink:
      • Minister of [insert portfolio name]
      • Represents the riding of [riding name]
      • Ministerial mandate letter
      • Ministerial briefing book
      • Ministerial appointments
  • Presentation
    • appears below the minister’s name

4: Biographical overview

Mandatory

Provides background information about the minister

  • Content
    • keep the text short and concise
    • must be consistent with the biographies presented on the Prime Minister’s website
  • Presentation
    • appears below the minister’s name, roles, and riding details

5: Contact information

Mandatory

Includes mailing, telephone and email contact information

  • Content
    • contact information includes email address, mailing address, telephone number and fax number
    • the email address is hyperlinked
  • Presentation
    • appears below the biographical overview
    • heading is labelled “Contact information”
    • telephone numbers:
      • the heading “Telephone:” precedes the telephone number
      • the heading “Fax:” precedes the fax number
      • telephone and fax numbers are displayed using the format 123-456-7890

6: Social media feeds widget

Optional

Features minister-specific social media channels

  • Content
    • use the Social media feeds widget pattern

7: Latest news

Conditional

Links to information from the news collection related to announcements made by the minister

  • Content
    • this component is mandatory, except when used for a parliamentary secretary
    • use the Latest news pattern
  • Presentation
    • appears below the contact information

8: Recent activities

Optional

Initiatives

Provide links to initiative pages relevant to the minister

Speeches

Provide links to recent speeches made by the minister from the news collection

Media advisories

Provide a list of recent media advisories related to upcoming events and announcements from the news collection

  • Content
    • consists of a series of headings that showcase links to content found in Canada.ca collections pertaining to the minister’s activities
    • a minimum of 2 headings must appear from the Canada.ca collections list:
      • initiatives
      • speeches
      • media advisories
      • other Canada.ca collections, as appropriate
    • the order of the headings must be consistent with this list
    • the heading label must be consistent with the Canada.ca collection name
    • a minimum of 2 links per heading is required
    • a maximum of 4 links per heading is allowed
  • Presentation
    • appears below the recent announcements
    • heading is labelled “Recent activities”
    • priority sequencing is from left to right

9: Gallery

Optional

Provides thumbnails hyperlinked to pictures and videos related to the minister

  • Content
    • it consists of a set of 4 hyperlinked images that link to pictures or videos related to the minister’s activities
    • link “All photos and videos” to a page displaying all photos and videos related to the minister
    • keep the hyperlink description text short and concise
  • Presentation
    • appears at the bottom of the main content area
    • heading is labelled “Gallery”
    • go to the Canada.ca GitHub page for image sizing details.

How to use the Parliamentary secretary modification

1b: Parliamentary secretary’s name

Mandatory

Provides the name of the parliamentary secretary

  • Content
    • must follow mandatory text format: [parliamentary secretary’s first and last names]
  • Presentation
    • the parliamentary secretary’s name must be a unique H1
    • must be the first component on the page

3b: Parliamentary secretary’s role(s)

Mandatory

Lists the official role(s) of the parliamentary secretary, with link(s) to related ministerial profile

  • Content
    • it must follow mandatory text format: Parliamentary Secretary to the Minister of [insert portfolio name]
    • the supported minister is hyperlinked to the appropriate ministerial profile page
    • when used for the head of a quasi-judicial arm’s-length institution, text must be the official title of the head without any hyperlink
  • Presentation
    • appears below the parliamentary secretary’s name
