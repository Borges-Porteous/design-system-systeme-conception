H1 subtitle : Alpha design pattern

This pattern is in alpha

Rationale

Many pages fit into a section of other pages. To make the context of the page clear, we often put that section in the title. For example:

How to apply - EI maternity and parental benefits

This is important because even though we want to focus on the "How to apply" part, each page needs a clear and unique title that stands alone.

We also put this in the H1, to make the purpose of the page sufficiently clear when people visit it.

This can make for some very long H1s, with repeated content across many of them that is only really useful to people the first time they get to the page.

Objective

Alternate layout that leaves the section visible, but emphasizes the current page.

When to use

Use when you would otherwise write the pagename, followed by a dash, followed by the section name.

How to implement

Page name - Section name
